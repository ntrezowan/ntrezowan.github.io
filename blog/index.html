How to Sync

One-time setup:

  • Go to github.com/settings/tokens
  • Click "Generate new token (classic)"
  • Name it "DevOps Toolkit Sync"
  • Check ONLY the gist scope
  • Generate token and copy it

In each tool:

  • Click "‚òÅÔ∏è Login"
  • Paste your token
  • Click "Login" (auto-pulls existing data)
  • Make changes, click "‚Üë Push" when done

On another device:

  • Login with the same token
  • Data auto-pulls from GitHub
  • Make changes, push when done

Tech Stack

  • Frontend: Vanilla JavaScript (no frameworks)
  • Storage: localStorage + GitHub Gists API
  • Styling: Gruvbox Dark theme, CSS Grid/Flexbox
  • Markdown: marked.js (Notes)
  • Syntax highlighting: highlight.js (Snippet)
  • Fonts: JetBrains Mono, Inter

Design Principles

  • Single-file apps - No build process, no dependencies
  • Offline-first - Works without internet (except sync)
  • No tracking - No analytics, no cookies, no third-party scripts
  • Fast - Instant load, no server round trips
  • Keyboard-driven - Shortcuts for common actions
  • Persistent - Data survives browser restarts
  • Private - Data stored locally and in your GitHub account only

Limitations

  • No real-time sync - Manual push/pull required
  • No conflict resolution - Last write wins
  • Don't edit on multiple devices simultaneously - One device at a time
  • localStorage limits - ~5-10MB per domain (browser-dependent)
  • No password manager - Don't store sensitive secrets in Config tool
